    About Ruby

    Ruby was created by Japanese programmer Yukihiro Matsumoto and first appeared in 1995. Ruby is dynamically typed and supports multiple paradigms. Matsumoto cites such langauges as Perl, Smalltalk, Ada, and Lisp as inspirations for Ruby. Matsumoto's stated objective was to be Lisp-like at its core, with object orientation inspired by Smalltalk, higher-order functions, and the practical utility of Perl.

    As always, the first program to write is the same in all languages: 

    puts "Hello, world!"

    This prints the string Hello, world! to the standard output.

    Ruby has been used for:

    • Scripting and automation, as a replacement for Perl and shell scripts
    • Mobile development through libraries such as RubyMotion
    • Desktop development, e.g. through Cocoa Ruby on the Macintosh
    • Web development, most famously through Ruby on Rails

    Comments in Ruby begin with the octothorpe (#), a convention common to Python and shell scripting.

    Conditionals and Loops

    Ruby sports three looping constructs:

    • for
    • while
    • until

    Ruby supports conditionals much like any other language, using these keywords:

    • if
    • elsif
    • unless
    • else

    Functions and Blocks

    Like all high-level languages, Ruby lets you define your own functions using the def keyword. 

    def say_hello_to (person)
    puts "Hello, #{person}."
end

    As you might expect, you can then call this function at will in your code. Parentheses around the arguments to a function are optional, and if a function has no arguments or a single argument, the parentheses are often omitted: 

    say_hello_to "William"       # prints "Hello, William."

    Classes

    Ruby is a pervasively object-oriented language; everything is an object. You can create your own classes from which you can instantiate your own objects. A very basic example of a class is given below: 

    class Doge
    attr_accessor :name

    def initialize(name="Doge")
        self.name = name
    end

    def such thing
        return "Such #{thing}"
    end

    def very thing
        return "Very #{thing}"
    end

    def wow
        return "Wow"
    end

    def i_am
        return self.name
    end
end

    Now we can create our very own doge: 

    irb(main):016:0> my_doge = Doge.new("Jeff")
=> #
irb(main):017:0> my_doge.wow
=> "Wow"
irb(main):018:0> my_doge.such "Ruby"
=> "Such Ruby"
irb(main):019:0> my_doge.very "program"
=> "Very program"
irb(main):020:0> my_doge.wow
=> "Wow"
irb(main):021:0> puts "I am #{my_doge.name}"
=> "I am Jeff"

    As you might guess, there's a lot more to Ruby classes than this silly example.
